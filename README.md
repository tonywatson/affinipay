# affinipay
Coding Exercise submission

Written using Ruby 2.7

The instructions assume you are using Ruby 2.7. You can validate your Ruby version by typing `ruby --version` in console.

1. Download `affinipay` project
2. From console `cd affinipay`
3. Run `ruby main.rb`

Input a time and the number of minutes to add to time to view result.
